# LFCS - Map of Content

> Linux Foundation Certified System Administrator

## Exam Overview

- **Duration:** 2 hours
- **Format:** 100% performance-based — live terminal, no multiple choice
- **Tasks:** 17–20 real-world sysadmin tasks
- **Pass mark:** 67%
- **Proctored:** Yes — online via PSI Bridge, taken from home in a browser
- **Cost:** $445 USD (includes one free retake within 12 months)
- **Validity:** 2 years (certifications earned after March 1, 2025)
- **Renewal:** Must retake and pass the full exam — no CEU credits or alternative paths

## On Exam Day

- Terminal only — no external internet access
- `man` pages and official Linux docs are allowed and available
- Time management is critical — skip and flag difficult tasks, return later
- Changes must **persist** after a reboot to count
